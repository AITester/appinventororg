Introduction to Canvases


Canvas -- a sub-panel of the screen

  • The app or a user can draw in it
  • The app can animate objects within it

Canvas has an X,Y coordinate system

  • X goes from 1 right to width of canvas
  • Y goes from 1 down to height of canvas (note down is bigger, unlike math)

Canvas properties, functions, and events

  • Canvas properties: Width, Height, BackgroundColor, PaintColor
  • Canvas functions: DrawCircle, DrawLine
  • Canvas events: Touched(x,y), Dragged(starts,starry,previousx,previousy, currentx, currenty)
  • note that dragged is triggered many times as user drags finger across the screen

Instructor Demo

  • Set the width of the canvas to "Fill parent"
  • Draw a circle at 30,30 not matter where touched
  • Draw circle at x,y on touched event
  • Note that event parameters of events found in "My Blocks": do not drag from event-handler
  • Show x,y coordinates in a label when you touch canvas, draw circle when touch canvas
  • Draw a circle in middle of canvas no matter where touched
