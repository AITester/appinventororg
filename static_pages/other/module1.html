Module 1  Introduction to Event-Driven Apps

The introduction acclimates students with the App Inventor environment, leads them to building their first apps, including HelloPurr and a painting app, and provides an introduction to event-handlers, function calls, conditionals, variables, and drawing canvases. The module culminates with a creative project in which the students create a soundboard app.

Programming concepts: events, event handlers, function calls, parameters, variables

Components: Button, Sound, Canvas
