Quiz 3

1.

(10) To what events does the MoleMash app respond?

Answer

The MoleMash app responds to 1) the user touching the screen, which triggers a Canvas.Touched event and sometimes an ImageSprite.Touched event, and 2) a Clock.Timer event-- the passing of time is an event.

2.

(10) What properties of an ImageSprite (or Ball) are used to record its location? What does each represent?

Answer
The location of an image sprite or ball is recorded in the X and Y properties, with X determining the horizontal location and Y the vertical location. The X and Y values are relative to the enclosing Canvas, so X=5 means the sprite is 5 pixels to the right of the left-edge of the Canvas. A Y value of 0 refers to the top of the Canvas, and Y values get bigger as you go down.

3.

(20) Explain why the subtractions are performed in the blocks below:

Answer
The subtractions in the MoveMole procedure are performed so that the image always stays completely on the Canvas. The X-Y location of an image sprite corresponds to its top-left corner, so if a large X or Y value is randomly selected, part of the image sprite might be to the right or below the canvas and thus not appear.

4.

(10) Assume a Canvas has imageSprite1 on it. When the user touches imageSprite1, what event are triggered?

Answer
When ImageSprite1 (or any image sprite) is touched by the user, two events are triggered: ImageSprite1.Touched and Canvas.Touched.
